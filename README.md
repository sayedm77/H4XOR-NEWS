# H4X0R-News App
Builded with SwiftUI.
It is a hackerNews reader to get hold of  the latest articles that are trending on hackingNews. 
============
## Features
* How to work with SwiftUI and identifiable protocol
* Navigation View 
* Observer Design pattern
* WebKit
